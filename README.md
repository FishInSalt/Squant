## 📋 项目概述

### 产品愿景
实现一套面向个人研究的量化交易系统，用于研究和验证交易策略。

### 目标用户
有一定量化交易经验的个人投资者，具备基础编程能力。

### 市场范围
- **初始阶段**：加密货币市场
- **后续扩展**：港股、美股

## 📦 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 行情看板 | 查看热门和自选加密货币行情 | P0 |
| 自定义策略 | 提供策略模板和接口规范 | P0 |
| 策略库 | 上传、校验、管理策略 | P0 |
| 策略运行 | 支持回测、模拟运行、实盘运行 | P0 |
| 多策略并发 | 同时运行多个策略，进程隔离 | P1 |
| 策略控制 | 随时停止策略和退出交易 | P1 |
| 运行监控 | 查看策略运行详情（行情、订单） | P2 |
| 账户配置 | 配置交易所API Key | P0 |
