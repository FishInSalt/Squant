## 6. 执行迁移

### 6.1 升级数据库

```bash
# 升级到最新版本
alembic upgrade head

# 升级到指定版本
alembic upgrade <revision_id>

# 升级+1版本
alembic upgrade +1
```

### 6.2 降级数据库

```bash
# 降级到上一个版本
alembic downgrade -1

# 降级到指定版本
alembic downgrade <revision_id>

# 降级到初始版本
alembic downgrade base
```

### 6.3 查看迁移历史

```bash
# 查看所有迁移版本
alembic history

# 查看当前版本
alembic current

# 查看即将执行的迁移
alembic upgrade head --sql
```
