## 📊 数据库架构

```
┌────────────────────────────────────────────────────────────┐
│                    PostgreSQL 16                           │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              TimescaleDB Extension                   │  │
│  │  ┌────────────────────────────────────────────────┐  │  │
│  │  │          Hyper Tables (时序超表)               │  │  │
│  │  │  • KlineData (K线数据)                         │  │  │
│  │  │  • OrderBook (订单簿)                          │  │  │
│  │  │  • TradeData (成交数据)                        │  │  │
│  │  └────────────────────────────────────────────────┘  │  │
│  └──────────────────────────────────────────────────────┘  │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              Standard Tables (标准表)                │  │
│  │  • Strategy (策略配置)                               │  │
│  │  • StrategyExecution (策略运行)                      │  │
│  │  • Order (订单)                                      │  │
│  │  • Position (持仓)                                   │  │
│  │  • BacktestResult (回测结果)                         │  │
│  └──────────────────────────────────────────────────────┘  │
├────────────────────────────────────────────────────────────┤
│                    SQLAlchemy ORM                          │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              Base Models & Mixins                    │  │
│  │  • Base ( declarative_base() )                       │  │
│  │  • TimestampMixin (created_at, updated_at)          │  │
│  └──────────────────────────────────────────────────────┘  │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              Domain Models                           │  │
│  │  • Strategy, KlineData, Order, Position, etc.       │  │
│  └──────────────────────────────────────────────────────┘  │
├────────────────────────────────────────────────────────────┤
│                    Alembic Migrations                     │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              Migration Scripts                       │  │
│  │  • versions/xxxxx_initial_schema.py                  │  │
│  │  • versions/xxxxx_add_indexes.py                    │  │
│  └──────────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────────┘
```

### 核心业务表

| 表名 | 说明 | 时序表 |
|------|------|--------|
| Strategy | 策略配置 | 否 |
| StrategyExecution | 策略运行记录 | 否 |
| KlineData | K线数据 | ✅ |
| OrderBook | 订单簿 | ✅ |
| TradeData | 成交数据 | ✅ |
