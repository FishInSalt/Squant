## 9. 数据库备份和恢复

### 9.1 备份数据库

```bash
# 全量备份
pg_dump -U squant -h localhost -d squant -F c -f squant_backup_$(date +%Y%m%d).dump

# 仅备份数据 (不含结构)
pg_dump -U squant -h localhost -d squant --data-only -F c -f squant_data_$(date +%Y%m%d).dump

# 仅备份结构 (不含数据)
pg_dump -U squant -h localhost -d squant --schema-only -F c -f squant_schema_$(date +%Y%m%d).dump
```

### 9.2 恢复数据库

```bash
# 恢复全量备份
pg_restore -U squant -h localhost -d squant -F c squant_backup_20240124.dump

# 恢复到新数据库
createdb -U squant -h localhost squant_new
pg_restore -U squant -h localhost -d squant_new -F c squant_backup_20240124.dump
```
