## 7. 部署架构设计

### 7.1 本地部署 (Docker Compose)

```yaml
# docker-compose.yml

version: '3.8'

services:
  # PostgreSQL数据库
  postgres:
    image: postgres:15-alpine
    container_name: squant-db
    environment:
      POSTGRES_USER: squant
      POSTGRES_PASSWORD: squant_password
      POSTGRES_DB: squant
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

  # Redis缓存
  redis:
    image: redis:7-alpine
    container_name: squant-redis
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data

  # FastAPI单体应用 (MVP)
  api:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: squant-api
    ports:
      - "8000:8000"
    environment:
      DATABASE_URL: postgresql://squant:squant_password@postgres:5432/squant
      REDIS_URL: redis://redis:6379
      DOCKER_HOST: unix:///var/run/docker.sock
    depends_on:
      - postgres
      - redis
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock  # Docker-in-Docker
      - ./strategies:/app/strategies

  # Prometheus
  prometheus:
    image: prom/prometheus:latest
    container_name: squant-prometheus
    ports:
      - "9090:9090"
    volumes:
      - ./config/prometheus.yml:/etc/prometheus/prometheus.yml
      - prometheus_data:/prometheus

  # Grafana
  grafana:
    image: grafana/grafana:latest
    container_name: squant-grafana
    ports:
      - "3000:3000"
    environment:
      GF_SECURITY_ADMIN_PASSWORD: admin
    volumes:
      - grafana_data:/var/lib/grafana

volumes:
  postgres_data:
  redis_data:
  prometheus_data:
  grafana_data:
```

### 7.2 云服务器部署

```markdown
# 部署文档

## 1. 本地部署

### 1.1 前置要求
- Docker >= 20.10
- Docker Compose >= 2.0
- Python >= 3.11

### 1.2 快速启动

```bash
# 1. 克隆仓库
git clone https://github.com/your-repo/squant.git
cd squant

# 2. 设置环境变量
cp .env.example .env
# 编辑 .env 文件,设置必要的配置

# 3. 启动所有服务
docker-compose up -d

# 4. 初始化数据库
docker-compose exec postgres psql -U squant -d squant -f /docker-entrypoint-initdb.d/init.sql

# 5. 访问Web界面
# http://localhost:8000
```

## 2. 云服务器部署

### 2.1 前置要求
- 云服务器 (Ubuntu 20.04+)
- 域名 (可选)
- SSL证书 (可选)

### 2.2 部署步骤

```bash
# 1. 安装Docker和Docker Compose
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 2. 克隆仓库
git clone https://github.com/your-repo/squant.git
cd squant

# 3. 配置环境变量
cp .env.example .env
nano .env  # 设置必要的配置

# 4. 启动服务
docker-compose up -d

# 5. 配置Nginx反向代理 (可选)
# 配置SSL证书 (Let's Encrypt)
```
