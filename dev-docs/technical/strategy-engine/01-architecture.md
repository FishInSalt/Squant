## 核心组件

| 组件 | 职责 |
|------|------|
| Process Manager | 管理策略进程的生命周期 |
| Strategy Process | 独立进程运行策略代码 |
| Sandbox | 代码安全隔离和资源限制 |
| Context | 提供数据和交易能力的抽象层 |

## 运行模式

| 模式 | 说明 | 特点 |
|------|------|------|
| Backtest | 回测模式 | 使用历史数据，模拟撮合 |
| Paper | 模拟交易 | 实时数据，虚拟账户 |
| Live | 实盘交易 | 实时数据，真实下单 |
