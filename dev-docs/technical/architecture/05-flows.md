## 4. 风控检查流程

```
订单请求
    │
    ▼
┌─────────────────┐
│  单笔限额检查   │──── 超限 ───▶ 拒绝订单
└────────┬────────┘
         │ 通过
         ▼
┌─────────────────┐
│  持仓比例检查   │──── 超限 ───▶ 拒绝订单
└────────┬────────┘
         │ 通过
         ▼
┌─────────────────┐
│  日亏损限额检查 │──── 超限 ───▶ 触发熔断
└────────┬────────┘
         │ 通过
         ▼
┌─────────────────┐
│  频率限制检查   │──── 超限 ───▶ 拒绝订单
└────────┬────────┘
         │ 通过
         ▼
    提交订单
```
