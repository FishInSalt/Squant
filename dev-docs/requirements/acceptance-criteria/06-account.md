# 验收标准 - 账户中心

## ACC-001: 添加交易所 API 配置

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 进入账户配置页 | 点击"添加账户" | 显示配置表单 |
| 2 | 填写完整信息 | 点击保存 | 配置保存成功 |
| 3 | 必填项未填 | 点击保存 | 提示必填项 |

---

## ACC-002: 支持 Binance 交易所

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 选择 Binance 交易所 | 填写 API Key 和 Secret | 可以保存配置 |
| 2 | 配置正确的 Binance API | 测试连接 | 连接成功 |
| 3 | 配置错误的 API | 测试连接 | 显示"认证失败"错误 |

---

## ACC-003: 支持 OKX 交易所

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 选择 OKX 交易所 | 填写 API Key、Secret、Passphrase | 可以保存配置 |
| 2 | 配置正确的 OKX API | 测试连接 | 连接成功 |
| 3 | Passphrase 未填写 | 点击保存 | 提示"Passphrase 为必填项" |

---

## ACC-004: API Key 加密存储

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 保存 API 配置 | 查看数据库 | API Key 和 Secret 以加密形式存储 |
| 2 | 加载配置时 | 系统使用 API | 能正确解密并使用 |
| 3 | 查看配置页面 | - | Secret 显示为掩码（如 ****） |

---

## ACC-005: API 连接测试

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 填写 API 配置 | 点击"测试连接" | 显示测试进度 |
| 2 | API 有效 | 测试完成 | 显示"连接成功"，显示账户余额 |
| 3 | API 无效 | 测试完成 | 显示"连接失败"及错误原因 |
| 4 | 网络超时 | 测试完成 | 显示"连接超时，请检查网络" |

---

## ACC-006: 编辑/删除 API 配置

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 已有 API 配置 | 点击"编辑" | 可以修改配置信息 |
| 2 | 已有 API 配置 | 点击"删除" | 显示确认对话框 |
| 3 | 该 API 有策略正在使用 | 尝试删除 | 提示"有策略正在使用此账户，无法删除" |

---

## ACC-010: 各交易所资产汇总

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 已配置交易所账户 | 进入资产页面 | 显示各交易所资产列表 |
| 2 | 资产列表 | - | 显示每个币种的数量和估值 |
| 3 | 多个交易所 | - | 显示总资产汇总 |
