# 验收标准 - 系统设置

## SYS-001: 手动下载历史数据

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 进入数据管理页 | 选择交易对和时间范围 | 可以发起下载任务 |
| 2 | 下载任务开始 | - | 显示下载进度 |
| 3 | 下载完成 | - | 数据可用于回测 |
| 4 | 下载失败 | - | 显示错误原因，支持重试 |

## SYS-010: 系统日志查看

| # | Given | When | Then |
|---|-------|------|------|
| 1 | 进入日志页面 | - | 显示最新日志列表 |
| 2 | 日志列表 | - | 每条日志显示时间、级别、模块、内容 |
| 3 | 日志很多 | 滚动加载 | 支持分页或无限滚动 |
