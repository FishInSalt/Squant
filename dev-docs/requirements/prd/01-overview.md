# PRD - 项目概述

## 1.1 背景

市面上现有的量化交易系统存在以下问题：
- 商业软件收费高昂，不适合个人研究使用
- 开源方案功能分散，缺乏完整的一站式解决方案
- 多数系统仅支持单一市场，扩展性差

因此，需要开发一套**免费、开源、可扩展**的个人量化交易系统。

## 1.2 项目目标

构建一套完整的量化交易系统，实现以下核心目标：

| 目标 | 描述 |
|------|------|
| **策略研究** | 提供完整的策略开发、回测、优化工具链 |
| **多市场支持** | 前期支持加密货币市场，后续扩展至港股、美股 |
| **灵活部署** | 支持本地和云服务器部署 |
| **易于使用** | 通过 Web 界面进行交互，降低使用门槛 |

## 1.3 目标用户

- **主要用户**: 开发者本人（单用户系统）
- **用户画像**: 具备编程能力，熟悉 Python，有量化交易研究需求

## 1.4 项目范围

### 1.4.1 Phase 1 (MVP) - 加密货币市场

| 范围 | 内容 |
|------|------|
| **交易所** | Binance、OKX |
| **交易类型** | 现货交易 |
| **核心功能** | 行情看板、策略管理、回测、模拟盘、实盘 |

### 1.4.2 Phase 2 - 功能增强

- 合约交易支持
- 更多加密货币交易所
- 高级风控功能
- 通知系统（Telegram、微信）

### 1.4.3 Phase 3 - 市场扩展

- 港股市场接入
- 美股市场接入
- 用户认证系统
