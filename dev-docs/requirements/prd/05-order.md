# PRD - 订单中心

## 2.5.1 订单状态定义

订单在系统中的生命周期状态：

| 状态 | 英文标识 | 说明 | 可流转至 |
|------|----------|------|----------|
| 待提交 | Pending | 策略生成，尚未发送到交易所 | 已提交、已取消 |
| 已提交 | Submitted | 已发送到交易所，等待撮合 | 部分成交、完全成交、已取消、已拒绝 |
| 部分成交 | PartialFilled | 部分数量已成交 | 完全成交、已取消 |
| 完全成交 | Filled | 全部数量已成交 | （终态） |
| 已取消 | Cancelled | 用户或系统取消 | （终态） |
| 已拒绝 | Rejected | 交易所拒绝 | （终态） |
