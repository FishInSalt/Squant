# PRD - 风控中心

## 2.6.1 风控规则

**功能描述**: 配置和管理风控规则。

**风控规则类型**:

| 规则类型 | 描述 | 触发动作 |
|----------|------|----------|
| **单笔限额** | 单笔订单最大金额/数量 | 拒绝下单 |
| **持仓限制** | 单个交易对最大持仓比例 | 拒绝下单 |
| **日亏损限制** | 当日最大亏损金额/比例 | 暂停策略 |
| **总亏损限制** | 策略运行以来最大亏损 | 停止策略 |
| **频率限制** | 单位时间内最大下单次数 | 延迟下单 |
| **波动熔断** | 价格短时间大幅波动 | 暂停策略 |

**功能要求**:

| 编号 | 需求描述 | 优先级 |
|------|----------|--------|
| RSK-001 | 配置单笔交易限额 | P0 |
| RSK-002 | 配置单交易对最大持仓比例 | P0 |
| RSK-003 | 配置每日最大亏损限制 | P0 |
| RSK-004 | 配置策略总亏损限制 | P0 |
| RSK-005 | 配置下单频率限制 | P1 |
| RSK-006 | 配置价格波动熔断 | P1 |
| RSK-007 | 风控规则开关 | P0 |
| RSK-008 | 风控触发记录 | P0 |

## 2.6.2 熔断机制

**功能描述**: 紧急情况下的自动保护机制。

| 编号 | 需求描述 | 优先级 |
|------|----------|--------|
| RSK-010 | 手动一键熔断（停止所有策略） | P0 |
| RSK-011 | 手动一键平仓（平掉所有持仓） | P0 |
| RSK-012 | 自动熔断条件配置 | P0 |
| RSK-013 | 熔断后通知 | P0 |
| RSK-014 | 熔断解除机制 | P1 |
