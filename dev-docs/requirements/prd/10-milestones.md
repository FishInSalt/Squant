# PRD - 里程碑规划

**目标**: 实现最小可用产品，满足基础策略研究需求。

---

**目标**: 搭建项目框架、开发环境和部署配置

**包含需求**: 无功能需求编号（基础设施）

- Docker 部署配置
- CI/CD 基础流程
