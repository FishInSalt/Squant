# PRD - 交易中心

## 2.4.1 回测运行

**功能描述**: 使用历史数据对策略进行回测验证。

**功能要求**:

| 编号 | 需求描述 | 优先级 |
|------|----------|--------|
| TRD-001 | 选择策略和交易对 | P0 |
| TRD-002 | 设置回测时间范围 | P0 |
| TRD-003 | 设置初始资金 | P0 |
| TRD-004 | 设置手续费率 | P0 |
| TRD-005 | 设置滑点模拟 | P1 |
| TRD-006 | 配置策略参数 | P0 |
| TRD-007 | 启动回测任务 | P0 |
| TRD-008 | 显示回测进度 | P0 |
| TRD-009 | 生成回测报告 | P0 |

**回测报告指标**:

| 指标类别 | 指标名称 | 说明 |
|----------|----------|------|
| **收益指标** | 总收益率 | 策略总收益百分比 |
| | 年化收益率 | 折算为年化的收益率 |
| | 基准收益率 | 买入持有策略的收益率 |
| | 超额收益 | 策略收益 - 基准收益 |
| **风险指标** | 最大回撤 | 最大净值回撤百分比 |
| | 最大回撤持续期 | 从回撤到恢复的最长时间 |
| | 波动率 | 收益率标准差（年化） |
| | 下行波动率 | 负收益的波动率 |
| **风险调整收益** | 夏普比率 | (收益率-无风险利率) / 波动率 |
| | 索提诺比率 | (收益率-无风险利率) / 下行波动率 |
| | 卡尔玛比率 | 年化收益率 / 最大回撤 |
| **交易统计** | 总交易次数 | 总成交笔数 |
| | 胜率 | 盈利交易 / 总交易 |
| | 盈亏比 | 平均盈利 / 平均亏损 |
| | 平均持仓时间 | 每笔交易平均持仓时长 |
| | 最大连续亏损 | 最大连续亏损次数 |
