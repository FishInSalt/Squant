# PRD - 附录

## 6.1 术语表

| 术语 | 说明 |
|------|------|
| K 线 / Candlestick | 表示一段时间内价格变动的图表形式 |
| OHLCV | Open, High, Low, Close, Volume 的缩写 |
| 回测 / Backtest | 使用历史数据验证策略 |
| 模拟盘 / Paper Trading | 使用虚拟资金进行模拟交易 |
| 实盘 / Live Trading | 使用真实资金进行交易 |
| 滑点 / Slippage | 预期成交价与实际成交价的差异 |
| 夏普比率 / Sharpe Ratio | 风险调整后收益指标 |
| 最大回撤 / Max Drawdown | 净值从峰值到谷值的最大跌幅 |
| 熔断 / Circuit Breaker | 异常情况下自动停止交易的机制 |

## 6.2 优先级说明

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，MVP 必须实现 |
| P1 | 重要功能，MVP 后优先实现 |
| P2 | 增强功能，有余力时实现 |
