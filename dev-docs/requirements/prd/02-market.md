# PRD - 行情中心

## 2.2.1 热门行情

**功能描述**: 展示当前市场热门加密货币的实时行情数据。

| 编号 | 需求描述 | 优先级 |
|------|----------|--------|
| MKT-001 | 展示 Top 20 热门交易对（按交易量排序） | P0 |
| MKT-002 | 显示实时价格、24h 涨跌幅、24h 交易量 | P0 |
| MKT-003 | 支持按交易所筛选（Binance / OKX / 全部） | P0 |
| MKT-004 | 实时价格更新（WebSocket 推送） | P0 |
| MKT-005 | 点击交易对进入详情页/K线图 | P1 |
